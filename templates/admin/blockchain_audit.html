How Blockchain Verification Works

When a patient's diagnosis is saved, we compute a SHA-256 hash of the record data (symptoms, disease, confidence, timestamp) combined with the previous record's hash — creating a chain. On audit, we recompute each hash and compare it to the stored hash. If they match → record is intact. If they differ → tampering detected.
